# Deploy 当前扩展工程到指定 Cocos 项目

将当前工程目录部署到目标 Cocos 项目的 extensions 目录下。

## 执行步骤

1. 读取当前工作目录作为源目录（`$CWD`），从 `package.json` 获取扩展名称（`name` 字段）
2. 询问用户目标 Cocos 项目根目录路径
3. 目标目录 = `{目标项目根目录}/extensions/{扩展名称}`
4. 删除目标目录（如存在）
5. 将源目录完整拷贝到目标目录
6. 验证拷贝结果（列出目标目录文件）

## 注意

- 源目录为当前命令执行时所在的工作目录，不要硬编码
- 扩展名称从当前目录的 `package.json` 中的 `name` 字段读取
- 目标项目路径需每次向用户确认，不要硬编码
